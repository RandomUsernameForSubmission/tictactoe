# Tic Tac Toe

A Python implementation of the classic Tic Tac Toe game with basic computer opponent.

## Installation

The only libraries used are 'tkinter' and 'random', which are standard Python libraries. Therefore, there are no additional dependencies to install.

## How to Run

1. Navigate to the project root directory
2. Execute the main script:

```bash
python main.py
```

## Gameplay

- Play against a computer opponent with random choices
- Click on the square where you want to add your pawn
- The computer plays immediately after your move
- Simply follow the instructions displayed in your screen
- Win by getting three marks in a row (horizontal, vertical or diagonal)
